# Installation et Construction

Découvrez comment installer et construire l'édition communautaire de Canyon.

## Configuration de l'Environnement

Avant de commencer l'installation, vous devez configurer les variables d'environnement.

```yaml
# Prisma Config
DATABASE_URL=postgres://username:password@localhost:5432/dbname
```

## Node.js

Clonez le projet

```shell
git clone https://github.com/canyon-project/canyon
```

Installez les dépendances et construisez le projet

```shell
pnpm install && pnpm run build
```

Modifiez le fichier .env et créez les tables de la base de données

```shell
pnpm run migrate
```

Exécutez le projet avec Node.js

```shell
node packages/canyon-backend/dist/main.js
```


## Docker

Pour exécuter Canyon avec Docker, il vous suffit de spécifier le chemin du fichier de variables d'environnement et le numéro de port

```shell
docker run -d -p 8080:8080 -v /your/path/.env:/app/.env zhangtao25/canyon:main
```

## Démarrage Réussi

Une fois le démarrage réussi, deux services principaux seront disponibles :

- Canyon frontend
- Canyon backend

Accédez à `http://localhost:8000/login` pour vous connecter à Canyon.
