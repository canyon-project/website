# Next.js

import { Steps } from "nextra/components";

Installation et configuration de Next.js.

<Steps>

  ### Créer un projet
  Commencez par créer un nouveau projet React `next.js` avec la commande suivante.

  ```sh npm2yarn copy
  npx create-next-app@latest
  ```

  ### Ajouter Canyon et sa configuration

  > [!TIP]
  >
  > Le compilateur Next.js récent utilise par défaut swc. Pour la configuration du plugin babel, consultez [ici](https://nextjs.org/docs/advanced-features/customizing-babel-config).


  ```sh npm2yarn copy
  npm install babel-plugin-istanbul babel-plugin-canyon -D
  ```

  ### Configurer le plugin babel

  ```js filename="babel.config.js" {3} copy
  module.exports = {
    presets: ["next/babel"],
    plugins: ["istanbul", "canyon"],
  };
  ```

  ### C'est tout
  Consultez la suite dans [Première donnée de couverture - Vérification](/docs/getting-started/first-coverage#verification).
</Steps>
