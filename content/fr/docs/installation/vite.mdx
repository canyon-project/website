Installation et configuration de Vite.

<Steps>

  ### Créer un projet
  Commencez par créer un nouveau projet React `vite` avec la commande suivante.

  > [!TIP]
  >
  > Actuellement, seul le modèle React de vite est pris en charge

  ```sh npm2yarn copy
  npm create vite@latest
  ```

  ### Ajouter Canyon et sa configuration

  ```sh npm2yarn copy
  npm install babel-plugin-istanbul babel-plugin-canyon -D
  ```

  ### Configurer le plugin babel

  ``` ts filename="vite.config.ts" copy {7,8,9}
  import { defineConfig } from 'vite'
  import react from '@vitejs/plugin-react'

  // https://vite.dev/config/
  export default defineConfig({
  plugins: [react({
      babel:{
        plugins:["istanbul","canyon"]
      }
    }))],
  })

  ```

  ### C'est tout
  Consultez la suite dans [Première donnée de couverture - Vérification](/docs/getting-started/first-coverage#verification).
</Steps>
