Installation et configuration de React Native.

<Steps>

  ### Créer un projet

  Documentation de [React Native](https://reactnative.dev/)

  ### Ajouter Canyon et sa configuration

  > [!TIP]
  >
  > Vous devez connaître [react-native-babel-preset](https://reactnative.dev/docs/javascript-environment#javascript-syntax-transformers).


  ```sh npm2yarn copy
  npm install babel-plugin-istanbul babel-plugin-canyon -D
  ```

  ### Configurer le plugin babel

  ```js filename="babel.config.js" {3} copy
  module.exports = {
    presets: ["module:@react-native/babel-preset"],
    plugins: ["istanbul", "canyon"],
  };
  ```

  ### C'est tout
  Consultez la suite dans [Première donnée de couverture - Vérification](/docs/getting-started/first-coverage#verification).
