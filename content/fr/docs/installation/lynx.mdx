Installation et configuration de Lynx.

<Steps>

  ### Créer un projet

  Documentation de [Lynx](https://lynxjs.org/)

  ### Ajouter Canyon et sa configuration

  > [!TIP]
  >
  > Le compilateur Lynx utilise par défaut swc. Documentation sur la configuration des plugins [ici](https://lynxjs.org/zh/api/rspeedy/rspeedy.tools.swc.html#toolsswc-property)


  ```sh npm2yarn copy
  npm install swc-plugin-coverage-instrument swc-plugin-canyon -D
  ```

  ### Configurer les plugins swc

  ```js filename="lynx.config.ts" {19,20} copy
  import { defineConfig } from '@lynx-js/rspeedy'
  import { pluginQRCode } from '@lynx-js/qrcode-rsbuild-plugin'
  import { pluginReactLynx } from '@lynx-js/react-rsbuild-plugin'

  export default defineConfig({
    plugins: [
      pluginQRCode({
        schema(url) {
          return `${url}?fullscreen=true`
        },
      }),
      pluginReactLynx(),
    ],
    tools:{
      swc:{
        jsc:{
          experimental:{
            plugins:[
                ['swc-plugin-coverage-instrument',{}],
                ['swc-plugin-canyon',{}]
            ]
          }
        }
      }
    }
  })

  ```

  ### C'est tout
  Consultez la suite dans [Première donnée de couverture - Vérification](/docs/getting-started/first-coverage#verification).
