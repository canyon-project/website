# Canyon Extension

Canyon Extension est une extension Chrome qui vous aide à collecter des données de couverture pendant vos tests manuels.

## Installation

Cliquez sur [Chrome Web Store](https://chromewebstore.google.com/detail/canyon-extension/omnpafdjidgpdmlimbangcjjaaodbeof) pour télécharger et installer Canyon Extension.

## Utilisation

1. Ouvrez la page de votre projet.
2. Cliquez sur le popup de Canyon Extension.
3. Configurez les options de rapport.
4. Commencez les tests.
