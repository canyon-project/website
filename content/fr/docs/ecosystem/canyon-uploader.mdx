# Canyon Uploader

Canyon Uploader est un outil binaire pour rapporter des données de couverture. Il propose des fichiers binaires pour les plateformes Linux, macOS et Windows.

La dernière version peut être téléchargée sur [GitHub Release](https://github.com/canyon-project/canyon/releases).

## Fonctionnalités

Lire et agréger les données de couverture du répertoire local `.canyon_output` puis les rapporter

## Utilisation

```bash
chmod +x ./canyon-uploader
canyon-uploader map --report-id=reportID
```
