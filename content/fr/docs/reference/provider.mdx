# Fournisseurs CI Supportés

import { Callout } from 'nextra/components'

<Callout type="info" emoji="ℹ️">
  Canyon peut prendre en charge n'importe quel fournisseur CI. Si votre fournisseur CI n'est pas dans la liste, veuillez nous le signaler à wr.zhang25@gmail.com.
</Callout>

Voici la liste des fournisseurs CI pris en charge par Canyon et leurs variables d'environnement :

| Fournisseur                                                    | ID du projet | SHA | Branche |
| ------------------------------------------------------------ | -- | -- | -- |
| [Azure Pipelines](https://azure.microsoft.com/en-us/services/devops/pipelines/) | BUILD_REPOSITORY_NAME | BUILD_SOURCEVERSION | BUILD_SOURCEBRANCH |
| [CircleCI](https://circleci.com/)                            | CIRCLE_REPOSITORY_URL | CIRCLE_SHA1 | CIRCLE_BRANCH |
| [Drone](https://drone.io/)                                   | DRONE_REPO | DRONE_COMMIT_SHA | DRONE_BRANCH |
| [Github Actions](https://github.com/features/actions)        | GITHUB_REPOSITORY | GITHUB_SHA | GITHUB_REF |
| [GitLab CI](https://about.gitlab.com/gitlab-ci/)             | CI_PROJECT_ID | CI_COMMIT_SHA | CI_COMMIT_BRANCH |
| [Jenkins](https://jenkins.io/)                               | GIT_REPOSITORY | GIT_COMMIT | GIT_BRANCH |
| [Travis CI](https://travis-ci.org/)                          | TRAVIS_REPO_SLUG | TRAVIS_COMMIT | TRAVIS_BRANCH |
