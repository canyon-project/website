Canyon(의미: "협곡", 발음: /ˈkænjən/)은 간단한 Babel 설정으로 JavaScript 커버리지 수집과 실시간 보고서 생성을 가능하게 하여 엔드투엔드 테스트에서의 커버리지 수집 문제를 해결합니다.

## 기능 특징

Canyon은 JavaScript 엔드투엔드 테스트 커버리지 수집을 위해 특별히 설계되었으며, 다음과 같은 기능 특징이 있습니다:

- [정확하고 효율적](/docs/core-concepts/separate-hit-and-map)：hit과 map의 분리를 통해 컴파일 시 초기 커버리지 데이터가 생성되어 정확하고 효율적인 커버리지 데이터 수집 가능
- [SourceMap](/docs/core-concepts/restore-source-code-coverage)：정확한 소스 코드 커버리지 복원
- [빌드 도구](/docs/installation/getting-started)：next.js, vite, webpack 등의 일반적인 빌드 도구 커버리지 솔루션 제공
- [자동화 프레임워크](/docs/end-to-end-testing/getting-started)：일반적인 UI 자동화 프레임워크 통합 솔루션
- [CI 제공업체](/docs/reference/provider)：일반적인 CI 제공업체(Github Actions, GitLab Runner)에 적응하고 CI 환경 변수 감지
