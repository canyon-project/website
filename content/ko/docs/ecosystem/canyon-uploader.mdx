# Canyon 업로더

Canyon 업로더는 커버리지 데이터를 보고하기 위한 바이너리 도구입니다. linux, macOS, windows 세 가지 플랫폼의 바이너리 파일을 제공합니다.

최신 버전은 [GitHub Release](https://github.com/canyon-project/canyon/releases)에서 다운로드할 수 있습니다.

## 기능

로컬 `.canyon_output` 디렉토리에 있는 커버리지 데이터를 읽어 집계하여 보고합니다.

## 사용법

```bash
chmod +x ./canyon-uploader
canyon-uploader map --report-id=reportID
```
