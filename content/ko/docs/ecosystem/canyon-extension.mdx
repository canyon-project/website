# Canyon Extension

Canyon Extension은 수동 테스트 과정에서 커버리지 데이터를 수집하는 데 도움을 주는 Chrome 확장 프로그램입니다.

## 설치

[Chrome Web Store](https://chromewebstore.google.com/detail/canyon-extension/omnpafdjidgpdmlimbangcjjaaodbeof)를 클릭하여 Canyon Extension을 다운로드하고 설치하세요.

## 사용법

1. 프로젝트 페이지를 엽니다.
2. Canyon Extension의 팝업을 클릭합니다.
3. 보고 옵션을 구성합니다.
4. 테스트를 시작합니다.
