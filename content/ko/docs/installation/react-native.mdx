  ### 프로젝트 생성

  React Native [문서](https://reactnative.dev/)

  ### Canyon 및 구성 추가

  > [!TIP]
  >
  > [react-native-babel-preset](https://reactnative.dev/docs/javascript-environment#javascript-syntax-transformers)에 대해 이해할 필요가 있습니다.


  ```sh npm2yarn copy
  npm install babel-plugin-istanbul babel-plugin-canyon -D
  ```

  ### babel 플러그인 구성

  ```js filename="babel.config.js" {3} copy
  module.exports = {
    presets: ["module:@react-native/babel-preset"],
    plugins: ["istanbul", "canyon"],
  };
  ```
