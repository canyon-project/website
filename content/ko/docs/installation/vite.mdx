# Vite

import { Steps } from "nextra/components";

Vite 설치 및 구성.

<Steps>

  ### 프로젝트 생성
  먼저 다음 명령을 사용하여 새 React 프로젝트 `vite`를 생성합니다.

  > [!TIP]
  >
  > 현재는 vite의 react 템플릿만 지원합니다.

  ```sh npm2yarn copy
  npm create vite@latest
  ```

  ### Canyon 및 구성 추가

  ```sh npm2yarn copy
  npm install babel-plugin-istanbul babel-plugin-canyon -D
  ```

  ### babel 플러그인 구성

  ``` ts filename="vite.config.ts" copy {7,8,9}
  import { defineConfig } from 'vite'
  import react from '@vitejs/plugin-react'

  // https://vite.dev/config/
  export default defineConfig({
  plugins: [react({
      babel:{
        plugins:["istanbul","canyon"]
      }
    })],
  })

  ```

  ### 완료
  계속해서 [첫 번째 커버리지 데이터 - 확인](/docs/getting-started/first-coverage#확인)을 참조하세요.
</Steps>
