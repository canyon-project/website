# 커스텀 데이터 보고

특정 상황에서, 특히 앱에 내장된 H5 페이지에서 커버리지 데이터를 보고해야 할 때, PC용 Canyon Chrome 확장 프로그램과 같은 도구가 없기 때문에 커버리지 데이터 보고 기능을 직접 구현해야 합니다.

## 적용 가능한 상황

- 앱에 내장된 H5 페이지
- Canyon Chrome 확장 프로그램을 사용할 수 없는 환경
- 커스텀 보고 로직이 필요한 상황
- 모바일 웹 애플리케이션

## 구현 방법

다음은 기본적인 커스텀 보고 구현 방법입니다:

```javascript copy
const coverageFirstValue = Object.values(window.__coverage__||{})[0];
if (coverageFirstValue) {
  fetch(coverageFirstValue.dsn,{
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      Authorization: `Bearer ${coverageFirstValue.reporter}`
    },
    body: JSON.stringify({
      coverage: window.__coverage__,
      ...coverageFirstValue
    })
  })
}
```
