# 설치 및 빌드

Canyon 커뮤니티 버전을 설치하고 빌드하는 방법을 알아보세요.

## 환경 구성

설치를 시작하기 전에 환경 변수를 구성해야 합니다.

```yaml
# Prisma Config
DATABASE_URL=postgres://username:password@localhost:5432/dbname
```

## Node.js

프로젝트 클론

```shell
git clone https://github.com/canyon-project/canyon
```

의존성을 설치하고 프로젝트 빌드

```shell
pnpm install && pnpm run build
```

.env 파일을 수정하고 데이터베이스 테이블 생성

```shell
pnpm run migrate
```

Node.js를 사용하여 프로젝트 실행

```shell
node packages/canyon-backend/dist/main.js
```


## Docker

Docker를 사용하여 Canyon을 실행하려면 환경 변수 파일 경로와 포트 번호만 지정하면 됩니다

```shell
docker run -d -p 8080:8080 -v /your/path/.env:/app/.env zhangtao25/canyon:main
```

## 성공적인 시작

성공적으로 시작한 후 두 가지 주요 서비스가 있습니다:

- Canyon frontend
- Canyon backend

`http://localhost:8000/login`에 접속하여 Canyon에 로그인하세요.
