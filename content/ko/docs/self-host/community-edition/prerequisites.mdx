# 전제 조건

자신의 인프라에 Canyon을 설치하기 위한 전제 조건입니다.

Canyon은 Docker 컨테이너 집합으로 패키징된 자체 호스팅 API 개발 플랫폼입니다. [Docker Engine](https://docs.docker.com/engine)을 실행할 수 있는任何운영 체제에서 Canyon을 설치하고 실행할 수 있습니다. 로컬 컴퓨터 또는 선택한 클라우드 공급자에서 Canyon을 사용할 수 있습니다.

## 시스템 요구 사항

Canyon은 소규모 및 대규모 배포에서 모두 잘 작동하도록 설계되었습니다. Canyon을 실행하기 위한 최소 요구 사항은 Docker를 지원하는 운영 체제와 빌드 이미지를 생성하기 위한 4개의 CPU 코어 + 4GB RAM, 그리고 생성된 출력 파일을 호스팅하기 위한 적어도 1개의 CPU 코어 + 2GB RAM입니다.

### Node.js、npm、pnpm 설치

#### Node.js + npm

[`Node.js`](https://nodejs.org/en) (v18+) 및 [`npm`](https://www.npmjs.com) (v9+)을 설치하세요.

- [Node.js + npm 설치 안내서](https://nodejs.org/en/download)


### pnpm

[`pnpm`](https://pnpm.io) (v9 권장)을 설치하세요.

- [pnpm 설치 안내서](https://pnpm.io/installation)

### Docker

[`Docker`](https://www.docker.com) (v20+)을 설치하세요.

- [Docker 설치 안내서](https://docs.docker.com/engine/install)

### Postgres 데이터베이스

Canyon은 기본 데이터를 저장하기 위해 Postgres 데이터베이스를 사용합니다(14.x 이상 필요). 로컬 또는 클라우드 공급자에 호스팅되는 원하는任何Postgres 데이터베이스 공급자를 사용할 수 있습니다. 다음 형식의 유효한 Postgres 데이터베이스 URL을 보유하고 있는지 확인하세요:

```
postgres://username:password@localhost:5432/dbname
```

#### 테이블 생성

```shell
pnpm run migrate
```

또는 직접 테이블 생성 [SQL](https://github.com/canyon-project/canyon/blob/main/packages/canyon-backend/prisma/create_table.sql)을 사용할 수 있습니다.
