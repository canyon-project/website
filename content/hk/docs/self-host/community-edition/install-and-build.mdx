# 安裝和構建

了解如何安裝和構建 Canyon 社區版。

## 配置環境

在開始安裝之前，您需要配置環境變量。

```yaml
# Prisma Config
DATABASE_URL=postgres://username:password@localhost:5432/dbname
```

## Node.js

克隆項目

```shell
git clone https://github.com/canyon-project/canyon
```

安裝依賴項並構建項目

```shell
pnpm install && pnpm run build
```

修改.env文件，創建數據庫表

```shell
pnpm run migrate
```

使用nodejs運行項目

```shell
node packages/canyon-backend/dist/main.js
```


## Docker

使用docker運行canyon只需要指定環境變量文件路徑和端口號即可

```shell
docker run -d -p 8080:8080 -v /your/path/.env:/app/.env zhangtao25/canyon:main
```

## 啟動成功

啟動成功後會有兩個主要的服務：

- Canyon frontend
- Canyon backend

訪問 `http://localhost:8000/login` 登錄 Canyon。
