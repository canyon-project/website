# 先決條件

在您自己的基礎架構上安裝 Canyon 的先決條件。

Canyon 是一個自託管 API 開發平台，打包為一組 Docker 容器。您可以在任何可以運行[Docker Engine](https://docs.docker.com/engine) 的操作系統上安裝和運行 Canyon 。您可以在本地計算機或您選擇的雲提供商上使用 Canyon。

## 系統要求

Canyon 的設計使其在小型和大型部署中都能良好運行。運行 Canyon 的最低要求是支持 Docker 的操作系統和 4 個 CPU 核心 + 4GB RAM 來生成構建映像，以及至少 1 個 CPU 核心 + 2GB RAM 來託管生成的輸出文件。

### 安裝 Node.js、npm、pnpm

#### Node.js + npm

安裝 [`Node.js`](https://nodejs.org/en) (v18+) 和 [`npm`](https://www.npmjs.com) (v9+).

- [Node.js + npm 安裝手冊](https://nodejs.org/en/download)


### pnpm

安裝 [`pnpm`](https://pnpm.io) (建議使用v9).

- [pnpm 安裝手冊](https://pnpm.io/installation)

### Docker

安裝 [`Docker`](https://www.docker.com) (v20+).

- [Docker 安裝手冊](https://docs.docker.com/engine/install)

### Postgres數據庫

Canyon 使用 Postgres 數據庫來存儲基礎數據（需要14.x以上）。您可以使用您選擇的任何 Postgres 數據庫提供商 - 託管在本地或雲提供商上。確保您擁有以下格式的有效 Postgres 數據庫 URL：

```
postgres://username:password@localhost:5432/dbname
```

#### 創建表

```shell
pnpm run migrate
```

或者直接使用建表 [SQL](https://github.com/canyon-project/canyon/blob/main/packages/canyon-backend/prisma/create_table.sql)。
