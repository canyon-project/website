<Steps>

  ### 創建項目

  React Native [文檔](https://reactnative.dev/)

  ### 添加 Canyon 及其配置

  > [!TIP]
  >
  > 您需要了解 [react-native-babel-preset](https://reactnative.dev/docs/javascript-environment#javascript-syntax-transformers) 。


  ```sh npm2yarn copy
  npm install babel-plugin-istanbul babel-plugin-canyon -D
  ```

  ### 配置 babel 插件

  ```js filename="babel.config.js" {3} copy
  module.exports = {
    presets: ["module:@react-native/babel-preset"],
    plugins: ["istanbul", "canyon"],
  };
  ```
