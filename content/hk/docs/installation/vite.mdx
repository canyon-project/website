# Vite

import { Steps } from "nextra/components";

安裝和配置 Vite 。

<Steps>

  ### 創建項目
  首先使用以下命令創建一個新的 React 項目 `vite` 。

  > [!TIP]
  >
  > 目前只支持 vite 的 react 模版

  ```sh npm2yarn copy
  npm create vite@latest
  ```

  ### 添加 Canyon 及其配置

  ```sh npm2yarn copy
  npm install babel-plugin-istanbul babel-plugin-canyon -D
  ```

  ### 配置 babel 插件

  ``` ts filename="vite.config.ts" copy {7,8,9}
  import { defineConfig } from 'vite'
  import react from '@vitejs/plugin-react'

  // https://vite.dev/config/
  export default defineConfig({
  plugins: [react({
      babel:{
        plugins:["istanbul","canyon"]
      }
    })],
  })

  ```

  ### 就是這樣
  繼續查看 [第一個覆蓋率數據 - 檢查](/docs/getting-started/first-coverage#%E6%A3%80%E6%9F%A5)。
</Steps>
