# Next.js

import { Steps } from "nextra/components";

安裝和配置 Next.js 。

<Steps>

  ### 創建項目
  首先使用以下命令創建一個新的 React 項目 `next.js` 。

  ```sh npm2yarn copy
  npx create-next-app@latest
  ```

  ### 添加 Canyon 及其配置

  > [!TIP]
  >
  > 最新next.js編譯器默認使用swc，有關babel插件的配置，請查看[這裡](https://nextjs.org/docs/advanced-features/customizing-babel-config)。


  ```sh npm2yarn copy
  npm install babel-plugin-istanbul babel-plugin-canyon -D
  ```

  ### 配置 babel 插件

  ```js filename="babel.config.js" {3} copy
  module.exports = {
    presets: ["next/babel"],
    plugins: ["istanbul", "canyon"],
  };
  ```

  ### 就是這樣
  繼續查看 [第一個覆蓋率數據 - 檢查](/docs/getting-started/first-coverage#%E6%A3%80%E6%9F%A5)。
</Steps>
