# 自定義數據上報

在某些場景下，特別是APP內嵌的H5頁面需要上報覆蓋率數據時，由於沒有類似PC端的Canyon Chrome插件，需要自定義實現覆蓋率數據的上報功能。

## 適用場景

- APP內嵌的H5頁面
- 無法使用Canyon Chrome插件的環境
- 需要自定義上報邏輯的場景
- 移動端Web應用

## 實現方案

以下是一個基本的自定義上報實現方案：

```javascript copy
const coverageFirstValue = Object.values(window.__coverage__||{})[0];
if (coverageFirstValue) {
  fetch(coverageFirstValue.dsn,{
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      Authorization: `Bearer ${coverageFirstValue.reporter}`
    },
    body: JSON.stringify({
      coverage: window.__coverage__,
      ...coverageFirstValue
    })
  })
}
```
