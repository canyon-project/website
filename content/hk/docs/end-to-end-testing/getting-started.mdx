# 入門

與單元測試不同的是，端到端測試會揭起瀏覽器，我們目標是收集瀏覽器內的覆蓋率數據。所以複雜度會相對較高。以下是大致流程。

- 暴露出瀏覽器的全局變量，在正確的時機把覆蓋率數據寫到本地
- 在UI自動化完成後使用 [canyon-uploader](/docs/ecosystem/canyon-uploader) 讀取本地 `.canyon_output` 目錄下的覆蓋率數據聚合上報。
- 將caseID關聯 [Report ID](/docs/core-concepts/report-id) ，保證case可回溯。
