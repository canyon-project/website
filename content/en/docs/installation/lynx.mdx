Install and configure Lynx.

<Steps>

  ### Create Project

  [Lynx](https://lynxjs.org/) documentation

  ### Add Canyon and Its Configuration

  > [!TIP]
  >
  > The Lynx compiler uses swc by default. Plugin configuration documentation [here](https://lynxjs.org/zh/api/rspeedy/rspeedy.tools.swc.html#toolsswc-property)


  ```sh npm2yarn copy
  npm install swc-plugin-coverage-instrument swc-plugin-canyon -D
  ```

  ### Configure swc Plugin

  ```js filename="lynx.config.ts" {19,20} copy
  import { defineConfig } from '@lynx-js/rspeedy'
  import { pluginQRCode } from '@lynx-js/qrcode-rsbuild-plugin'
  import { pluginReactLynx } from '@lynx-js/react-rsbuild-plugin'

  export default defineConfig({
    plugins: [
      pluginQRCode({
        schema(url) {
          return `${url}?fullscreen=true`
        },
      }),
      pluginReactLynx(),
    ],
    tools:{
      swc:{
        jsc:{
          experimental:{
            plugins:[
                ['swc-plugin-coverage-instrument',{}],
                ['swc-plugin-canyon',{}]
            ]
          }
        }
      }
    }
  })

  ```
